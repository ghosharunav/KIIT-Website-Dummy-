KIIT Dummy Website
A dummy university website inspired by Kalinga Institute of Industrial Technology built using HTML, CSS, and PHP.
This project was created as a practice project for learning full-stack web development basics, including frontend design and simple backend authentication using PHP.

Project Overview

The website simulates a basic university portal where users can navigate through different sections such as academics, admissions, and contact information. It also includes separate login sections for Admin and Students with password-based access to demonstrate simple backend functionality.
The goal of this project was to understand:
1.Website layout and UI design
2.Navigation systems
3.Basic authentication using PHP
4.Integration of frontend and backend components
